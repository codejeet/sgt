Proposal

Modulo — Bootstrapped Go‑To‑Market Plan

Focus: close the first 3–5 paying pediatric clinics quickly, using cold calling + a 14‑day trial, while shipping the minimum product/security work needed to sell credibly.
North star
Reduce check‑in paperwork + staff rework via SMS mobile intake packets
Fastest channel
Cold calling → 15‑min demo → 14‑day trial

1) ICP

  • Primary: independent pediatric practices (1–5 providers), 1–3 locations.
  • Buyer: practice owner + practice manager.
  • Pitch: “text forms before visits, track completion, fewer missing forms.”

2) Pricing recommendation

  • Start with one number: $199/mo per location (up to 5 providers), +$49/provider beyond 5.
  • Trial: 14‑day free trial, only after kickoff call. Don’t give it free to everyone.
  • Founding clinics: lock their rate for first N clinics to create urgency.

3) Development steps needed (sellable MVP)

Must‑have (2–4 weeks)
  • Multi‑tenancy (tenant isolation, tenant admin)
  • Stripe subscriptions + trials + dunning
  • HIPAA baseline: encryption, access controls, logs, BAAs/policies in progress
Then (conversion/retention)
  • Pediatric packet templates library
  • Dashboard polish (resend flows, completion analytics)
  • Template cloning + “save as template”

HIPAA messaging stance

  • Do not claim “HIPAA compliant” today.
  • State current baseline clearly: encryption in transit/at rest, role-based access controls, audit logging.
  • State open work clearly: complete BAA coverage, finalized policies/runbooks, formal third-party assessment.

4) Bootstrapped acquisition (ranked)

  • Cold calling (best): list → call practice manager → book demo.
  • Cold email (follow-up after calls).
  • Partnerships: billing consultants, practice manager communities.

5) Cold call script (Jeremy Miner-inspired)

Permission opener → problem questions → book demo. Full script is in COLD_CALL_SCRIPT.md. Excerpt:

“Did I catch you in the middle of something?”
“Can I take 20 seconds to tell you why I called, then you tell me if it’s worth scheduling something?”
“How are you handling intake today—clipboards/scanning or digital forms?”

6) Best plan of action

  • Lead with operational relief (not “young developer asking advice”).
  • Use a trial with a concrete success criterion.
  • Instrument the funnel: dials/day, demos/week, time-to-first-packet, completion rate.
